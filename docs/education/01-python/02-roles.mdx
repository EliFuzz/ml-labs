          </ul>

        <td>

        </td>

        </td>
      </tr>

          <b>Database Expertise</b>

        </td>
      </tr>
      <tr>

          <b>Data Pipeline Focus</b>
        </td>
        <td>

        </td>
        <td>Low: Sometimes uses simple automations for data scraping</td>
        <td>Medium: Processes data for modeling, sometimes builds pipelines</td>
      </tr>
      <tr>
        <td>
          <b>Machine Learning/AI</b>
        </td>
        <td>Rarely involved directly, may deploy or support ML models</td>
        <td>
          Rarely creates ML models, may use predictive analytics/simpler
          techniques
        </td>
        <td>
          Highly involved: experiments, builds, and deploys ML and deep learning
          models
        </td>
      </tr>
      <tr>
        <td>
          <b>Visualization Skills</b>
        </td>
        <td>Low: Occasional, for pipeline monitoring</td>
        <td>

        </td>
        <td>

        <td>
          <b>Business Interaction</b>
        </td>
